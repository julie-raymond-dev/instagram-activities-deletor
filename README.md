# üì± **InstaDel - Instagram Activity Cleaner**

`instdel.py` is a simple yet powerful Python script that helps you clean up your Instagram activity, including comments, likes, reels, and posts. Built with Selenium and ChromeDriver, this script interacts with your Instagram account and allows you to quickly delete various types of activity. üöÄ

## ‚ú® **Features**

* üìù **Delete Comments**

### üì¶ **Installation**

